# Я проектирую своё благополучие

**Telegram-бот** — цифровой наставник для современной молодёжи, который помогает поддерживать баланс трёх составляющих благополучия:  
Физическое (Тело) · Психологическое (Душа) · Социальное (Развитие)

Проект выполнен в рамках открытого межрегионального хакатона **«Код здорового человека»**  
Партнёр трека: Кафедра философии с курсами культурологии, биоэтики и отечественной истории СИБГМУ

### Особенности проекта
- Полное соответствие ТЗ (3 блока, трекеры, SOS, навигатор помощи, дневник настроения, Pomodoro, профориентация)
- Геймификация: очки, ачивки, уровни
- Хранение данных пользователя (SQLite)
- Дружелюбный Tone of Voice (на «ты», эмодзи)
- Работает на Python 3.12–3.13 + aiogram 3.13.1

### Стек технологий
- Python 3.12+
- aiogram 3.13.1
- SQLite (встроен в Python)
- python-dotenv
- Docker + docker-compose (одной командой)

### Установка и запуск

#### Вариант 1 — Локально (классический)
```bash
# 1. Клонируем репозиторий
git clone https://github.com/bloodbath88/wellbeing-bot.git
cd wellbeing-bot

# 2. Создаём виртуальное окружение
python -m venv env
env\Scripts\activate        # Windows
# source env/bin/activate    # macOS/Linux

# 3. Устанавливаем зависимости
pip install -r requirements.txt

# 4. Создаём .env файл (скопировать из .env.example и вставить свой токен)
cp .env.example .env
# отредактируй .env → вставь свой BOT_TOKEN от @BotFather

# 5. Инициализируем базу данных
python init_db.py

# 6. Запускаем бота
python main.py


#### Вариант 2 - Docker-compose
# Просто выполни в папке проекта:
docker-compose up --build
