# wellbeing-bot
Чат-бот для хакатона "Код здорового человека" — баланс тела, души и развития
# Я проектирую своё благополучие

Telegram-бот — цифровой наставник по балансу трёх составляющих благополучия:  
Физическое (Тело) · Психологическое (Душа) · Социальное (Развитие)

Хакатон **«Код здорового человека» 2025**  
Трек: Чат-бот

## Функциональность (полностью по ТЗ)
- 3 блока: Тело / Душа / Развитие
- Трекер воды и сна
- SOS (дыхательная практика)
- Навигатор помощи (буллинг, родители, экзамены)
- Дневник настроения
- Pomodoro-таймер
- Мини-тест профориентации
- Геймификация: очки, ачивки, статистика
- Дружелюбный Tone of Voice (на «ты» + эмодзи)

## Стек технологий
- Python 3.12
- aiogram 3.13.1
- SQLite
- Docker + docker-compose

## Как запустить

### Вариант 1 — Локально
```bash
git clone https://github.com/твой_ник/wellbeing-bot.git
cd wellbeing-bot
python -m venv env
env\Scripts\activate        # Windows
pip install -r requirements.txt
cp .env.example .env        # вставь свой токен
python init_db.py
python main.py
